## Handwriting recognition

В данном репозитории реализована модель, определяющая с точностью 98%, рукописаные буквы. Используется [датасет](https://storage.yandexcloud.net/academy.ai/A_Z_Handwritten_Data.csv) по рукописным буквам английского языка. Архитектура модели - 2 простых полносвязных слоя (**Dense**). Использование графических ускорителей необязательно.

**Точность на тестовом образце:** 0.98

**Потери на тестовом образце:** 0.05
