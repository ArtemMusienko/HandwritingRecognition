## Handwriting recognition

Используется [датасет](https://storage.yandexcloud.net/academy.ai/A_Z_Handwritten_Data.csv) по рукописным буквам английского языка. Архитектура модели - 2 простых полносвязных слоя (**Dense**).

**Точность на тестовом образце:** 0.98

**Потери на тестовом образце:** 0.05
