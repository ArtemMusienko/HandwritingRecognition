## Handwriting recognition

В данном репозитории реализована модель, определяющая рукописаные буквы английского алфавита. Используется данный [датасет](https://storage.yandexcloud.net/academy.ai/A_Z_Handwritten_Data.csv). Архитектура модели - 2 простых полносвязных слоя (**Dense**). 

Результат работы кода: 

 - *точность* на тестовом образце: 0.98 
 - *потери* на тестовом образце: 0.05

В финальной части представлены графики и выводы по обученной модели, а также сделаны предсказания по 20 случайным изображениям  рукописных букв.

> Использование **графического ускорителя T4** необязательно.
